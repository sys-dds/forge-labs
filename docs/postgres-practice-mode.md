# PostgreSQL Practice Mode

Practice mode loads one independent feature clinic. Use it after the matching chapter idea is familiar.

```bash
./scripts/forge-test-postgres-practice.sh 02-postgres/_practice/01-idempotency-key-upsert-clinic
```

Inspect the scenario, starting point, solution, expected result, and break/fix drill. Then remove the relevant constraint, policy, function branch, trigger guard, or conflict target and rerun the proof.
