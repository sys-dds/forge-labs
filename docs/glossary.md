# Glossary

- **Candidate**: A row that is eligible for a later decision, such as ranking, filtering, or matching.
- **Constraint**: A database rule that protects correctness even when application code races.
- **Feed candidate**: A post selected by visibility and relationship rules before ranking.
- **Graph query**: A query over relationships such as follows, blocks, mutes, or mutual connections.
- **Keyset cursor**: A pagination cursor based on stable ordered columns, usually `created_at` plus `id`.
- **Mutual match**: Two users who both expressed a LIKE toward each other.
- **Proof test**: SQL that raises an exception when the chapter concept is broken.
