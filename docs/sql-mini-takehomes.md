# SQL Mini Take-Homes

Mini take-homes are short backend SQL assessments. Load the dataset, read the candidate task, write the query, then compare with the proof and rubric.

```bash
./scripts/forge-list-takehomes.sh
./scripts/forge-load-takehome.sh 01-sql/_takehomes/01-social-feed-takehome
./scripts/forge-test-takehome.sh 01-sql/_takehomes/01-social-feed-takehome
./scripts/forge-test-all-takehomes.sh
```

Use the review rubric to judge correctness, proof mindset, exclusion handling, count accuracy, and explanation quality.
