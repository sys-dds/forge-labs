# Capstone: marketplace/feed analytics interview query

Scenario: Marketplace feed analytics for seller treatment.

You will query `sellers, listings, orders, listing_events`. The important trap rows are 101 high quality boost; 102 popular risky suppress; 103 cold-start allow; 104 high impressions poor booking watch; 105 blocked listing excluded; 106 no recent impressions. The solution must pass both the base and variant seeds, so do not hard-code answer IDs.
