# SQL Mini Take-Homes

Mini take-homes are small company-style SQL assessments. They are not puzzles: each one has a named dataset, a candidate task, exact proof tests, a review rubric, follow-up questions, and a model walkthrough.

Run one:

```bash
./scripts/forge-test-takehome.sh 01-sql/_takehomes/01-social-feed-takehome
```

Run all:

```bash
./scripts/forge-test-all-takehomes.sh
```
